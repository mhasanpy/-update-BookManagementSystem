# 📚 Book Management System (Updated)

[![Python Version](https://img.shields.io/badge/python-3.7%2B-blue)](https://python.org)
[![License](https://img.shields.io/badge/license-MIT-green)](LICENSE)

A simple command-line **Book Management System** written in Python.  
Allows you to **add, view, search, update, and delete** books from a local JSON database.

---

## 🚀 Features

- ➕ Add new books (title, author, year, ISBN)
- 📋 List all books
- 🔍 Search books by title or author
- ✏️ Update book details
- ❌ Delete a book entry
- 💾 Persistent storage using `books.json`
